# AI Usage

## Tools Used

The following AI tools were used as assistance during the development of this technical exercise:

- ChatGPT

---

## Purpose of AI Assistance

AI was used as a support tool to help with:

- Structuring documentation files
- Generating initial examples of Playwright tests
- Reviewing test structure and automation approach
- Formatting Markdown documentation

The AI assistance was used primarily to speed up repetitive tasks and documentation formatting.

---

## Prompts / Instructions Used

Examples of instructions given to the AI tool included:

- Generate examples of Playwright API tests
- Suggest documentation structure for QA test strategy
- Format Markdown files for GitHub readability
- Provide examples of smoke test scenarios

---

## Human Review and Adjustments

All AI-generated content was reviewed and adjusted manually.

The following tasks were performed manually:

- Selection of testing strategy
- Prioritization of test coverage
- Definition of critical business flows
- Implementation and organization of test files
- Adjustments to Playwright configuration
- Adaptation of tests to the project structure

---

## Final Notes

AI was used as an assistant to accelerate documentation and provide example snippets, but the testing approach, structure and prioritization decisions were made manually.

The objective was to demonstrate practical QA automation skills while maintaining transparency regarding the use of AI tools.
